TRACE Protocol

Open standard for auditable autonomous systems — Action → Policy → Evidence

Get Started GitHub

Overview

TRACE is an open standard for auditable autonomous systems. It provides a unified model for recording actions, applying policies, and collecting evidence across different AI agents and platforms.

Core Loop: Action → Policy → Evidence 
// 1. Propose an action
const decision = await trace.propose({
  type: "send_email",
  actor: { kind: "agent", name: "mail-bot" },
  target: "mailto:client@acme.com",
  params: { subject: "Proposal" }
});

// 2. Policy evaluation (enforce or observe)
if (decision.status === "requires_approval") {
  await fetch(`/approve/${decision.actionId}`, { method: "POST" });
}

// 3. Record evidence
await trace.evidence(decision.actionId, [
  { name: "email_sent", pass: true, note: "msgId=123" }
]);

Core Concepts

Action

What an agent intends to do.

{
  "id": "a_123",
  "type": "send_email",
  "actor": { "kind": "agent", "name": "mail-bot" },
  "target": "mailto:client@acme.com",
  "params": { "subject": "Hi" },
  "timestamp": "2025-10-10T12:34:56Z"
}

Policy

Rules that must hold.

rules:
  - when: { actionType: "send_email" }
    require: ["reviewer_approval"]
    mode: enforce

Evidence

Proof of outcomes.

{
  "actionId": "a_123",
  "checks": [{ "name": "reviewer_approval", "pass": true, "approver": "@fernando" }],
  "timestamp": "2025-10-10T12:40:00Z"
}

Quickstart

1. Start the reference server

cd server
npm install && npm run dev   # http://localhost:8787

2. Install SDK

TypeScript:

npm install @trace/client

Python:

pip install trace-client
